Conversational AI Widget Demo

Embedding Instructions

Add this code to your website:

<iframe
    src="https://voice-convo-widget-futur-intel.replit.app/embed?apiKey=YOUR_API_KEY&agentId=YOUR_AGENT_ID&theme=%7B%22primary%22%3A%22%230066cc%22%2C%22background%22%3A%22%23ffffff%22%2C%22text%22%3A%22%23ffffff%22%7D"
    style="
        width: 400px;
        height: 600px;
        position: fixed;
        bottom: 20px;
        right: 20px;
        border: none;
        border-radius: 12px;
        box-shadow: 0 4px 20px rgba(0,0,0,0.2);
        z-index: 9999;
    "
    allow="microphone"
></iframe>

Features

  • Voice-based conversational AI using ElevenLabs API
  • Customizable appearance
  • Terms and conditions compliance
  • Real-time conversation status indicators

Configuration Parameters

  • apiKey: Your ElevenLabs API key
  • agentId: Your ConvAI agent ID
  • theme: Customize the appearance (optional)

Theme Configuration

Default theme values:

{
  "primary": "#0066cc",
  "background": "#ffffff",
  "text": "#ffffff"
}
