Chat Widget Implementation Guide

How to Add the Chat Widget to Your Website

Add the following code snippet to your website:


<!-- Chat Widget -->
<script>
    window.CONVAI_CONFIG = {
        widgetUrl: "https://{your-middleware-url}/widget.js",
        apiKey: "your-api-key",  // Get this from your dashboard
        agentId: "your-agent-id" // Your assigned agent ID
    };
</script>
<script src="https://{your-middleware-url}/widget.js" async></script>

Required Configuration:

  • apiKey: Your API key from the dashboard
  • agentId: Your assigned agent ID

Optional: Custom Element Implementation

Alternatively, you can use the widget as a custom element:


<voice-convo-widget 
    api-key="your-api-key"
    agent-id="your-agent-id"
    theme='{"primary":"#0066cc","background":"#ffffff","text":"#ffffff"}'
></voice-convo-widget>

Theme Customization

You can customize the widget's appearance using the theme attribute:

  • primary: Main color for buttons and accents
  • background: Chat window background color
  • text: Text color for buttons
