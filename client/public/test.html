SiteHQ Assistant Implementation Guide

Embedding Instructions

Add this code to your website:

<iframe
    src="/embed?apiKey=YOUR_API_KEY&agentId=YOUR_AGENT_ID&theme=%7B%22primary%22%3A%22%235c078c%22%2C%22background%22%3A%22%23ffffff%22%2C%22text%22%3A%22%23333333%22%7D"
    style="
        width: 400px;
        height: 600px;
        position: fixed;
        bottom: 20px;
        right: 20px;
        border: none;
        border-radius: 12px;
        box-shadow: 0 4px 20px rgba(0,0,0,0.2);
        z-index: 9999;
    "
    allow="microphone"
></iframe>

Configuration Parameters

  • apiKey: Your API key
  • agentId: Your agent ID
  • theme: Customize the appearance (optional)

Theme Configuration

Default theme values:

{
  "primary": "#5c078c",
  "background": "#ffffff",
  "text": "#333333"
}
