Widget Implementation Guide

How to Embed the Widget

Add the following code to your website:



<script>
    window.CONVAI_SETTINGS = {
        apiKey: "vc_live_8f4a9c2e7b6d5x3y1w9v8u4t2p0n7m5k3j1h",
        agentId: "your-agent-id",
        title: "Your Assistant Name"
    };
</script>


<script src="https://{your-replit-url}/widget.js" async></script>

Required Parameters:

  • apiKey: Your middleware API key (get this from your dashboard)
  • agentId: Your ConvAI agent ID
  • title (optional): Custom title for the chat window

Or use as a Custom Element:


<voice-convo-widget 
    api-key="vc_live_8f4a9c2e7b6d5x3y1w9v8u4t2p0n7m5k3j1h"
    agent-id="your-agent-id"
    theme='{"primary":"#0066cc","background":"#ffffff","text":"#ffffff"}'
></voice-convo-widget>
