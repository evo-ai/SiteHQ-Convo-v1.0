Widget Implementation Guide

How to Embed the Widget

Add the following code to your website:



<script>
    window.CONVAI_SETTINGS = {
        apiKey: "sk_d30f51b33804638dd5e2af1f942f1685ccacd0d95ef30500",
        agentId: "FnTVTPK2FfEkaktJIFFx",
        title: "LifePod Agent"
    };
</script>


<script src="https://voice-convo-widget-futur-intel.replit.app/widget.js" async></script>

Required Parameters:

  • apiKey: Your ElevenLabs API key
  • agentId: Your ConvAI agent ID
  • title (optional): Custom title for the chat window

Or use as a Custom Element:


<voice-convo-widget 
    api-key="sk_d30f51b33804638dd5e2af1f942f1685ccacd0d95ef30500"
    agent-id="FnTVTPK2FfEkaktJIFFx"
    theme='{"primary":"#0066cc","background":"#ffffff","text":"#ffffff"}'
></voice-convo-widget>
