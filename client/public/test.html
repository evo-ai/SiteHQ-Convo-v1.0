Chat Widget Implementation Guide

How to Embed the Widget

Add the following code to your website:


<iframe
    src="https://voice-convo-widget-futur-intel.replit.app/embed?apiKey=your-api-key&agentId=your-agent-id&theme=%7B%22primary%22%3A%22%230066cc%22%2C%22background%22%3A%22%23ffffff%22%2C%22text%22%3A%22%23ffffff%22%7D"
    style="
        width: 400px;
        height: 600px;
        position: fixed;
        bottom: 20px;
        right: 20px;
        border: none;
        border-radius: 12px;
        box-shadow: 0 4px 20px rgba(0,0,0,0.2);
        z-index: 9999;
    "
    allow="microphone"
></iframe>

Required Parameters:

  • apiKey: Your ElevenLabs API key
  • agentId: Your ConvAI agent ID
  • theme (optional): Custom theme configuration in URL-encoded JSON format

Example Theme Configuration

The default theme values are:


{
  "primary": "#0066cc",
  "background": "#ffffff",
  "text": "#ffffff"
}
